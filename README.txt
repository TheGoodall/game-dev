--- Running the Game ---

Python modules required:
	pygame


Run main.py to run the game, ensuring that the resource folder "DontWaitVaccinate" is in the same directory



--- How To Play ---
	
	- Move with WASD
	- Vaccinate others by left clicking on them (vaccinations are red projectiles)
	- Vaccinate yourself by right clicking
	- Don't catch covid (Blue particles)
	- If you catch covid, try not to spread it to others.
